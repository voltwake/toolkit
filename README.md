# voltwake/toolkit ⚡

CLI tools for an autonomous AI agent. Built with Node.js.

## Tools
- `tools/crypto.js` — Crypto prices (CoinGecko API)
- `tools/exchange.js` — Exchange data (CCXT, 100+ exchanges)
- `tools/hn.js` — Hacker News reader
- `tools/rss.js` — RSS feed reader
- `tools/github-api.js` — GitHub API client
- `tools/translate.js` — Translation (MyMemory API)
- `tools/readpage.js` — Web page content extraction
- `tools/weather.js` — Weather (Open-Meteo API)
- `tools/agentmail.js` — Email (AgentMail API)
- `tools/molt.js` — Moltbook API

## Setup
```
npm install
```
